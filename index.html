This tool works entirely in your browser with no server required. You can use it:

  • Directly from your local filesystem (file:// protocol)
  • As a saved HTML file on your desktop
  • Offline - no internet connection needed
  • For checking offline static sites and local HTML files

Note: External URL checking requires internet access. For advanced features, use the CLI version.

üìñ How to Use

Checking Links in Browser

  1. Click "Upload HTML Files" and select one or more HTML files from your computer
  2. Configure options (external URL checking, show valid links, etc.)
  3. Click "Check Links" to analyze
  4. View results grouped by file

Command Line Interface

For more advanced features, use the Python CLI:

  • python3 unresolver.py . - Check all HTML in current directory
  • python3 unresolver.py --site-root /path/to/root . - Specify site root for absolute URLs
  • python3 unresolver.py --index-files index.html,default.html . - Custom index files
  • python3 unresolver.py --help - Show all CLI options

This tool checks HTML files for broken links in the following tags:

  • <a href> - Hyperlinks
  • <link> - Stylesheets and resources
  • <img src> - Images
  • <script src> - JavaScript files
  • <iframe src> - Embedded content
  • <area href> - Image map areas

Note: This interface requires running a local web server. See README for CLI usage.
